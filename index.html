PhantomCalc

多功能计算器系统

项目简介

表面是普通计算器，实则是具有进程保护机制的"陷阱"工具，兼具实用性和娱乐性。

创新特性

  • 隐形守护模式（双进程互相守护架构）
  • Alt+Esc安全终止热键和窗口置顶功能
  • 强大的数学引擎（支持复数矩阵运算和微积分符号计算）
  • 单位换算系统（60+单位）
  • 历史记录云同步（AES-256加密存储）
  • 伪装模式（"计算错误"诱饵界面）

示例代码

# 进程保护机制
protection_thread = threading.Thread(target=protect_process)
protection_thread.start() # 后台守护进程

应用场景

数学教学工具、隐私保护、系统安全演示

LynxEngine

极致精简的浏览器引擎

项目简介

基于PyQt5和QtWebEngine构建的轻量级浏览器核心，提供基本网页浏览功能，适用于嵌入式系统开发。

核心特性

  • 极简架构设计（仅400行代码实现核心功能）
  • 完整导航功能（前进/后退/刷新/主页）
  • 智能URL处理（自动添加协议前缀）
  • 低资源占用（最小化系统消耗）
  • 可定制界面（模块化设计便于扩展）
  • 跨平台支持（Windows/Linux/macOS）

示例代码

# 创建浏览器实例
self.browser = QWebEngineView()
self.browser.setUrl(QUrl("https://www.baidu.com"))

# 导航功能实现
def navigate_to_url(self):
    url = self.url_bar.text()
    if not url.startswith(("http://", "https://")):
        url = "https://" + url

应用场景

嵌入式设备、教学演示、轻量级应用集成、浏览器功能研究
